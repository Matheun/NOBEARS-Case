## Setup

Make sure to install the dependencies:

```bash
# pnpm
pnpm install
```

Create a .env file with the following content:

```bash
APP_URL=http://localhost:3001
API_URL=https://data.pro-cluster.com
```

## Development Server

Start the development server on `http://localhost:3001`:

```bash
# pnpm
pnpm dev
```
